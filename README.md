# Peta_SexRatio_Perempuan_Watumalang
Peta ini memberikan informasi jumlah perempuan, dan luas wilayah di Kecamatan Watumalang. Sumber data diperoleh dari Kecamatan Watumalang Dalam Angka yang diterbitkan Badan Pusat Statistik Kecamatan Watumalang pada tahun 2025 yang sudah disimbolisasi.
